# AWS SAA-C03 Anki Deck Requirements

Generate a comprehensive Anki flashcard deck for the AWS Certified Solutions Architect - Associate (SAA-C03) exam following these specifications:

### 1. Content Scope & Alignment
- **Quantity:** Exactly 600 flashcards.
- **Accuracy:** Source information from the latest AWS documentation and whitepapers.
- **Curriculum Alignment:** Ensure content parallels major AWS training courses (e.g., Adrian Cantrill, Stephane Maarek).

### 2. Organization & Sequencing
- **Logical Flow:** Order cards chronologically according to a standard certification learning path (e.g., Global Infrastructure -> IAM -> EC2 -> S3 -> VPC -> Databases -> Serverless -> Security).
- **Topic Progression:** Introduce fundamental concepts before advanced architectures.

### 3. Metadata & Tagging
- **Categorization:** Tag every card with its relevant AWS service (e.g., `S3`, `Lambda`) and exam domain (e.g., `Design-Resilient-Architectures`).
- **Tag Hierarchy:** Use tags that mirror the logical order of service introduction in professional courses.

### 4. Format
- **Structure:** Provide the output in a format ready for Anki import (e.g., CSV with `Front`, `Back`, and `Tags` columns).
